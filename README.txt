This is a custom module template for use in Drupal 7.

Steps to get this on your development system:

1. Have Drupal 7 already installed and working on your system.

2. From your Drupal root folder navigate to sites/all/modules.

3. Create a folder named "custom" and go into this folder.

4. Navigate to this custom folder in git.

5. in git, issue the pull command - git pull
